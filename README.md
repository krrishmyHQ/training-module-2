# myHQ — Starter Kit · Module 1

Sales L&D onboarding program hosted on GitHub Pages.

## Pages
- `landing.html` — Entry / registration page (set as homepage)
- `index.html` — Course hub (all 3 chapters)
- `CRE.html` — Chapter 1: India CRE Overview
- `originstory.html` — Chapter 2: myHQ Origin Story
- `terminologies.html` — Chapter 3: CRE Terminologies

## GitHub Pages Setup
1. Push all files to a repo (e.g. `myhq-starterkit`)
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be at `https://<username>.github.io/myhq-starterkit/landing.html`

## Notes
- No backend required — all state uses `localStorage`
- No quiz or unlock codes — all chapters are open
- Email restriction removed for senior management demo
